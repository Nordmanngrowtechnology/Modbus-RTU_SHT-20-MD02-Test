# Read data with Modbus-RTU 485
### Modbus-RTU 485 Temp & Humidity sensor MD02

# Required 

- PC
- Modbus 485 serial to USB
- Sensor XY-MD02
- Voltage 12V DC
- Wiring

# Install
Install python and add python extension  **MinimalModbus**

`pip install -U minimalmodbus`



# Usage

Navigate open command line tool and navigate to project folder


# XY-MD02 or MD02
- Thea can use UART with Windows direct with a c++ script
